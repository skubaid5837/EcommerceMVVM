# EcommerceMVVM

EcommerceMVVM is an Android application built using Kotlin and MVVM architecture. 
It provides an e-commerce experience with product listings, detail pages, cart management, and modern Android development practices. 

---

## Features

- Product listing and details screens  
- Shopping cart management  
- User authentication and account management  
- Modern UI built using XML layouts and Material Components  
- ViewModel-driven state management  
- Repository pattern for data handling  
- Offline-first support (if integrated with Room or cache)  
- Jetpack Navigation for screen transitions  

---

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/skubaid5837/EcommerceMVVM.git
```

### 2. Open in Android Studio
- Use Android Studio Ladybug or newer
- Import as a Gradle project

### 3. Configure Dependencies
- Sync Gradle to install required dependencies
- Configure API keys or local data sources if applicable

### 4. Build & Run
- Choose an emulator or physical device
- Click Run in Android Studio

## Learning Highlights

This project demonstrates:

- Implementation of MVVM with Kotlin  
- Use of ViewModel and LiveData/Flow for state management  
- Repository pattern for data abstraction  
- Integration of Material Components for UI consistency  
- Use of dependency injection with Hilt  
- Modular, clean, and maintainable architecture
